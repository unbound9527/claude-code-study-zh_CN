# 子代理：同时处理多任务

## 子代理是什么

分身术，同时做多件事：

```
普通：Task A → 等 → Task B → 等 → Task C

子代理：Task A ──┐
         Task B ──┼── 同时 → 汇总
         Task C ──┘
```

## 什么时候用

| 场景 | 示例 |
|------|------|
| 准备演讲 | 写稿、做 PPT、准备 Q&A |
| 开店规划 | 市场分析、财务预算、选址 |
| 内容创作 | 文案、标题、配文 |

## 句式

```
请同时帮我处理：
1. [任务 A]
2. [任务 B]
3. [任务 C]

完成后整合成...
```

## 限制

| 能做 | 不能做 |
|------|--------|
| 独立任务 | 相互依赖的任务 |
| 同一领域多个问题 | 需要来回讨论的创意 |
