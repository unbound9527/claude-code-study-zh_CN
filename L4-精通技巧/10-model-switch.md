# 模型选择与切换

## 三种模型

| 模型 | 速度 | 成本 | 适合 |
|------|------|------|------|
| **Opus** | 慢 | 高 | 复杂分析、深度推理 |
| **Sonnet** | 中 | 中 | 日常任务、平衡之选 |
| **Haiku** | 快 | 低 | 简单问答 |

## 选择建议

| 任务 | 推荐 |
|------|------|
| 复杂代码重构 | Opus |
| 写代码、做项目 | Sonnet |
| 简单翻译/问答 | Haiku |
| 写重要报告 | Opus |

## 切换命令

| 命令 | 作用 |
|------|------|
| `/model` | 查看当前模型 |
| `/model claude-opus-4-6` | 切换到 Opus |
| `/model claude-haiku-4-6` | 切换到 Haiku |
| `/model auto` | 自动选择 |

## 成本优化

```
简单问题 → Haiku
日常任务 → Sonnet
复杂问题 → Opus
```
