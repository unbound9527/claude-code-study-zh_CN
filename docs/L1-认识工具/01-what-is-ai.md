# 什么是 AI 和 Claude Code？

## 本章目标

- 理解什么是 AI（人工智能）
- 知道 Claude Code 是什么
- 理解它的能力和局限性

---

## 什么是 AI？

### 用大白话说

AI（人工智能）就像一个超级聪明的机器人助手。它看过海量的书、网页、对话，所以它懂很多很多事情。

**但它不是真的"智能"**，它更像是：

- 一个记忆力超强的图书馆管理员
- 一个读过全世界所有作文的学生

### 比喻理解


| AI 就像  | 为什么             |
| ------ | --------------- |
| 超级图书馆  | 记得所有看过的内容       |
| 厉害的复读机 | 根据学过的内容组织回答     |
| 聪明的鹦鹉  | 会模仿、会学习、但不懂真正含义 |


---

## 什么是 Claude Code？

Claude Code 是 Anthropic 公司开发的 AI 对话工具。

你可以把它理解为一个**在线的超级助手**，它能：

- 回答问题
- 帮你写作
- 写代码、调试程序
- 分析数据
- 翻译内容
- ...几乎任何文字相关的事

### Claude Code 和其他 AI 有什么不同？


| 特点   | Claude Code | 普通搜索引擎 | 其他 AI 聊天 |
| ---- | ----------- | ------ | -------- |
| 交互方式 | 像聊天         | 关键词匹配  | 对话       |
| 理解能力 | 懂意思         | 只认字    | 懂意思      |
| 个性化  | 可以记住你       | 不能     | 有的可以     |
| 记忆对话 | 可以          | 不行     | 有的可以     |


---

## Claude Code 能做什么？

### 它擅长的事


| 类别   | 示例           |
| ---- | ------------ |
| 回答问题 | "解释一下相对论"    |
| 写作帮手 | "帮我写一封请假邮件"  |
| 学习辅导 | "帮我理解这道数学题"  |
| 翻译   | "把这段英文翻译成中文" |
| 编程   | "教我写一个网页"    |
| 分析   | "帮我分析这份数据"   |


### 它原本不擅长的事（有了工具后可以克服）


| 类别   | 原因（过去）          | 现在有了工具后            |
| ---- | --------------- | ----------------- |
| 实时信息 | 训练数据有截止日期     | 可以通过 WebFetch 获取实时信息 |
| 数学计算 | 大数字计算容易出错      | 可以调用计算器或代码精确计算     |
| 个人记忆 | 每次对话不记得上次说的   | 可以用 Memory 功能记住    |
| 找文件   | 不知道文件在哪       | 可以用 Glob/Grep 搜索文件  |

**关键认知**：AI 本身的知识有截止日期，但通过**工具调用**可以突破这些限制。Claude Code 的强大之处正在于此——它不仅仅是一个聊天机器人，而是一个可以通过工具"感知世界、执行动作"的 Agent 系统。
