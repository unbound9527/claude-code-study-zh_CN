# 什么是 AI 和 Claude Code？

## AI 是什么

AI（人工智能）不是真的"智能"，更像：

- 记忆力超强的图书馆管理员
- 读过全世界文章的复读机

它根据学过的内容组织回答，不是有真正的理解。

## Claude Code 是什么

Anthropic 开发的 AI 对话工具，通过工具调用可以"感知世界、执行动作"。

| 能力 | 说明 |
|------|------|
| 回答问题 | 知识问答、解释概念 |
| 写作帮手 | 写邮件、文章、代码 |
| 编程 | 写代码、调试、分析 |
| 翻译 | 中英互译 |
| 分析 | 数据、文本、代码分析 |

## Claude Code vs 其他工具

| | Claude Code | 搜索引擎 | 其他 AI |
|--|------------|---------|--------|
| 交互 | 对话 | 关键词 | 对话 |
| 理解 | 语义 | 字面 | 语义 |
| 个性化 | 可记忆 | 否 | 部分可以 |

## 有了工具后能做的事

| 原本不擅长 | 解决方案 |
|-----------|---------|
| 实时信息 | WebFetch 获取 |
| 精确计算 | Bash/计算器 |
| 记住信息 | Memory 功能 |
| 找文件 | Glob/Grep |

**核心**：工具调用让 Claude Code 不只是聊天机器人，而是 Agent 系统。
