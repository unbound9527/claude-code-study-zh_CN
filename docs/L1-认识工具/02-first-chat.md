# 第一次对话

## 本章目标

- 学会打开和关闭 Claude Code
- 掌握输入和发送消息的方法
- 能读懂 AI 的回复

---

## 打开 Claude Code

### 方法一：网页版

1. 打开浏览器（Chrome、Edge、Safari 等）
2. 输入 Claude Code 的网址
3. 登录你的账号

### 方法二：桌面应用

1. 下载 Claude Code 应用
2. 安装并打开
3. 登录账号

---

## 界面介绍

```
┌─────────────────────────────────────┐
│  Claude Code                        │

```

### 主要区域

| 区域 | 说明 |
|------|------|
| 对话区域 | 显示你和 AI 的对话历史 |
| 输入框 | 输入你的问题或需求 |
| 发送按钮 | 点击发送消息 |

---

## 第一次对话

### 第一步：输入问题

在输入框中输入：

```
你好，帮我解释一下什么是光合作用
```

### 第二步：发送

- 按键盘的 **Enter** 键
- 或点击发送按钮

### 第三步：等待回复

AI 会思考几秒钟，然后显示回复。

### 第四步：读懂回复

AI 的回复可能会包含：
- 直接回答问题
- 解释概念
- 举例子
- 问你是否需要更多信息

---

## 语音输入

如果你不想打字，可以使用语音输入！

### 使用方法

**Windows**
1. 点击输入框
2. 按住 `Win + H`
3. 开始说话

**macOS**
1. 点击输入框
2. 按住 `Fn + .` 或点击麦克风图标
3. 开始说话

### 语音输入技巧

| 技巧 | 说明 |
|------|------|
| 说清楚 | 吐字清晰，语速适中 |
| 简短停顿 | 每句话之间稍作停顿 |
| 标点符号 | 可以说"逗号""句号"添加标点 |
| 错误修正 | 说"删除"或"重新说"修正 |

---

## 关闭对话

### 方法一：直接关闭

关闭浏览器标签页或应用窗口。

### 方法二：开始新对话

- 点击"新建对话"或"New Chat"按钮
- 这不会删除之前的对话历史

### 方法三：退出登录

- 点击设置
- 选择"退出登录"

3. 比较两种方式的识别准确度

## 小测验

**1. 发送消息的方式有几种？**
- A. 1 种
- B. 2 种（Enter 或按钮）
- C. 3 种

**2. 语音输入的正确说法是？**
- A. Win + S
- B. Win + H（Windows）
- C. Ctrl + V

**3. 开始新对话会怎样？**
- A. 删除所有历史
- B. 保留历史，开启新对话
- C. 无法开始新对话

---
