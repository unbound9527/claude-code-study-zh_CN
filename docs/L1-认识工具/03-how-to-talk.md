# 怎么和它说话

## 基本格式

```
你：输入问题或需求
AI：回复
你：追问或新问题
...
```

像发微信，但**不**自动记住之前说的（除非开记忆功能）。

## 好问题 vs 坏问题

| 坏问题 | 好问题 | 为什么 |
|--------|--------|--------|
| "帮我看看" | "检查这段文字语法" | 太模糊 |
| "怎么做" | "做番茄钟 APP，告诉我步骤" | 缺背景 |
| "那个事" | "上周写的邮件再发一遍" | 指代不清 |

## 提问三法则

**1. 说清楚要什么**
```
❌ 帮我写东西
✅ 100字内写朋友圈文案，推荐新耳机
```

**2. 给背景**
```
❌ 这个方案怎么样
✅ 大学生校园卖零食，定价8元/包进价5元，合理吗
```

**3. 指定格式**
```
❌ 列几个学习方法
✅ 5个高效学习法，用表格，含方法名/步骤/适用场景
```

## 和聊天的区别

| 聊天 | Claude Code |
|------|-------------|
| 朋友记得上次说的 | 默认只记当前对话 |
| 朋友会猜你意思 | AI 只能按你说的理解 |
| 朋友会纠正你 | AI 尽量回答你 |

## 多窗口管理

**场景**：不同项目、不同话题、整理资料

**操作**：
- 新建：点击 New Chat
- 改名：悬停 → 编辑
- 删除：悬停 → 删除
- 搜索：左侧搜索框

**命名建议**：
```
❌ 对话1
✅ 英语学习-语法问题
✅ 论文第二章修改
```
