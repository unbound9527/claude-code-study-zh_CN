# Agent Teams：组建专家团队

## 是什么

像现实中的顾问团，请不同专家从各自角度给建议。

```
你需要决策
  ↓
组建团队：市场专家 + 财务专家 + 技术专家
  ↓
整合成完整建议
```

## vs 子代理

| | 子代理 | Agent Teams |
|--|--------|-------------|
| 像 | 一个人同时做多事 | 团队开会 |
| 适合 | 同类任务并行 | 不同专家协作 |

## 什么时候用

| 场景 | 需要的专家 |
|------|-----------|
| 创业决策 | 市场、财务、运营 |
| 买房决策 | 房产、律师、财务 |
| 职业规划 | 行业顾问、HR、职业教练 |
| 产品开发 | 市场、技术、用户体验 |

## 句式

```
我要 [目标]，请组建专家团队：
- [专家1]：负责 [职责]
- [专家2]：负责 [职责]
- [专家3]：负责 [职责]

整合成 [格式]
```

## 示例

```
开奶茶店，请组建顾问团队：
- 市场分析师：竞争店铺、目标客户
- 财务顾问：启动资金、每月成本、回本周期
- 运营专家：日常流程、人员配置
- 营销专家：开业活动、会员体系

整合成创业计划书。
```

## 注意

- 3-5 个专家足够
- 职责清晰，避免重叠
- 指定输出格式
