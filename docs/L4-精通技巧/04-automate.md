# 自动化工作流

## 自动化是什么

让重复的事自动完成，节省时间。

```
每天手动：整理笔记 → 生成待办 → 发消息 = 30分钟
自动化后：一键完成
```

## 什么时候自动化

重复越多，价值越高。

| 场景 | 频率 | 价值 |
|------|------|------|
| 每日日报 | 每天 | 高 |
| 周报 | 每周 | 高 |
| 偶尔翻译 | 按需 | 中 |

## 常用模板

**日报生成**：
```
根据信息生成日报：
[粘贴信息]
格式：完成事项 / 进行中 / 下周计划 / 备注
```

**会议记录**：
```
整理会议，提取：主要问题、决定、待办(负责人+截止)
[粘贴记录]
```

## 设计步骤

1. 识别重复任务
2. 提取不变部分（格式）
3. 做成模板

## 省力技巧

- 保存常用模板到笔记软件
- 批量处理：周五一次性生成下周5天日报
- 预设角色："我是数学老师，以后按这个身份帮我..."
