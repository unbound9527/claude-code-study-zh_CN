# 记忆功能

## 本章目标

- 理解什么是记忆功能
- 能设置和更新个人偏好
- 能有效利用记忆提升效率

---

## 什么是记忆功能？

Claude Code 有两种记忆：

| 类型 | 说明 | 例子 |
|------|------|------|
| 对话记忆 | 当前对话内记住 | 对话结束就忘 |
| 长期记忆 | 跨对话记住 | 下次也记得 |

**记忆功能 = 让 AI 长期记住关于你的信息**

---

## 为什么要用记忆功能？

没有记忆：
```
你：我叫李明
AI：你好李明

（新对话）
AI：你好，有什么帮助？
你：我是谁？
AI：不知道，你没告诉我
```

有记忆：
```
你：我叫李明，是产品经理
AI：好的李明，我记住了

（新对话）
AI：你好李明！有什么可以帮你的？
```

## 设置记忆

### 一次性设置

```
请记住以下关于我的信息：
- 我的名字：[名字]
- 我的职业：[职业]
- 我的写作风格偏好：[偏好]
- 我希望：[其他要求]
```

### 分次添加

```
另外，请记住：
- 我喜欢被叫：[称呼]
- 我常用的邮件签名是：[签名]
```

---

## 常见记忆内容

### 基本信息

| 类型 | 示例 |
|------|------|
| 名字 | "叫我小王" |
| 身份 | "我是一名大三学生" |
| 职业 | "我是高中数学老师" |

### 偏好设置

| 类型 | 示例 |
|------|------|
| 回答风格 | "我喜欢简洁的回答" |
| 写作风格 | "邮件要正式，但不要太生硬" |
| 语言 | "用中文回答，除非我要求翻译" |

### 工作/学习背景

| 类型 | 示例 |
|------|------|
| 项目 | "我在做一个电商 App" |
| 专业 | "我学的是金融" |
| 目标 | "我在准备托福考试" |

---

## 查看和更新记忆

### 查看记忆

```
请问你现在记住关于我的哪些信息？
```

### 更新记忆

```
请更新一下：
- [旧信息] → [新信息]
```

### 删除记忆

```
请删除 [某个记忆]
```

---

## 跟做

1. 对 AI 说：
```
请记住以下关于我的信息：
- 我的名字是 [你的名字]
- 我是 [身份]
- 我喜欢 [写作风格]
```

2. 关闭对话，开启新对话

3. 问："还记得我是谁吗？"

## 小测验

**1. Claude Code 什么时候会忘记你说的？**
- A. 永远不会忘
- B. 关闭对话后
- C. 只有你让它忘记才会忘

**2. 记忆功能的主要作用是？**
- A. 让 AI 记得所有对话
- B. 省去每次重复说明的麻烦
- C. 让 AI 更聪明

**3. 更新记忆应该说？**
- A. "请更新"
- B. "请记住新的：[新内容]"
- C. "忘记 [内容]"

---

## 答案

1-B | 2-B | 3-B
