# 省钱技巧

## 计费方式

按 **Token** 计费（1中文 ≈ 1-2 Token）。

## 省钱原则

| 原则 | 说明 |
|------|------|
| 精简问题 | 问清楚但不啰嗦 |
| 管理上下文 | 避免重复说同样的话 |
| 选择合适模型 | 不同任务用不同模型 |
| 批量处理 | 减少重复请求 |

## 技巧

**精简提问**：
```
❌ "我是大学生，学计算机，有道题不会..."
✅ "Python实现二叉树中序遍历，含注释"
```

**选择模型**：
| 模型 | 速度 | 成本 | 适合 |
|------|------|------|------|
| Opus | 慢 | 高 | 复杂分析 |
| Sonnet | 中 | 中 | 日常任务 |
| Haiku | 快 | 低 | 简单问答 |

**上下文管理**：
- 定期总结要点
- `/compact` 压缩历史
- 话题转换开新对话

**避免无效迭代**：
```
❌ 问"帮我写代码" → "重来" → "还不对"
✅ 一次说清楚：输入、输出、要求
```
