# 子代理：同时处理多任务

## 本章目标

- 理解什么是子代理
- 能用子代理同时处理多个任务
- 知道子代理的适用场景

---

## 什么是子代理？

**子代理 = 分身术**

正常情况下，你问一个问题，AI 顺序回答。

子代理可以让你**同时做多件事**：

```
普通模式：
Task A → 等待完成 → Task B → 等待完成 → Task C

子代理模式：
Task A ──┐
Task B ──┼── 同时进行 → 汇总结果
Task C ──┘
```

---

## 什么时候用子代理？

| 场景 | 示例 |
|------|------|
| 准备演讲 | 同时写稿、做 PPT、准备 Q&A |
| 开店规划 | 同时做市场分析、财务预算、选址分析 |
| 学习备考 | 同时复习多个科目 |
| 内容创作 | 同时写文案、写标题、写配文 |

---

## 怎么用子代理？

### 句式

```
请同时帮我处理以下任务：
1. [任务 A]
2. [任务 B]
3. [任务 C]
```

### 进阶句式

```
请同时帮我处理以下任务，完成后整合成一份文档：

1. [任务 A] → 格式：[格式]
2. [任务 B] → 格式：[格式]
3. [任务 C] → 格式：[格式]
```

---

## 示例

### 示例一：准备演讲

```
我要做一场"如何高效学习"的演讲，请同时帮我：
1. 写一份 10 分钟的演讲稿（约 1500 字）
2. 制作 PPT 大纲（10 页，每页要点）
3. 准备 5 个观众可能问的问题和答案

完成后整合成一份演讲准备文档。
```

### 示例二：开咖啡馆

```
我想开一家咖啡馆，请同时帮我：
1. 做市场分析（目标客户、竞争对手、选址建议）
2. 做财务预算（启动资金、每月开支、预期回本时间）
3. 制定开业前 3 个月的推广计划

最后整合成一份简明商业计划书。
```

---

## 子代理的限制

| 能做 | 不能做 |
|------|--------|
| 独立任务 | 相互依赖的任务 |
| 同一领域的多个问题 | 需要来回讨论的创意 |
| 格式不同的输出 | 一个任务的输出影响另一个 |

---

## 跟做

1. 输入：
```
请同时帮我做三件事：
1. 给我一个番茄炒蛋的食谱
2. 给我一个红烧肉的食谱
3. 给我一个清炒时蔬的食谱

每个食谱格式：材料 + 步骤（3-5 步）
```

2. 观察是否同时完成

## 练一练

设计一个需要同时处理的场景，用子代理完成：

1. 准备一次班级活动（同时：时间表、预算、任务分配）
2. 准备求职（同时：简历、cover letter、面试准备）
3. 准备旅行（同时：行程、预算、行李清单）

## 小测验

**1. 子代理的主要特点是？**
- A. 一个人做所有事
- B. 多个任务同时进行
- C. 依次完成任务

**2. 以下哪个适合用子代理？**
- A. 写完第一章再写第二章
- B. 同时做三道不同科的练习题
- C. A 和 B 都不适合

**3. 子代理不能处理什么类型的任务？**
- A. 独立任务
- B. 相互依赖的任务
- C. 简单任务

---

## 答案

1-B | 2-B | 3-B
