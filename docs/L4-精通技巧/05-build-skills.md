# 构建自己的技能

## 本章目标

- 理解什么是自定义技能
- 能创建个人专属的 Claude Code 配置
- 学会打造自己的 AI 工作环境

---

## 什么是自定义技能？

**自定义技能 = 为特定场景定制的 AI 配置**

就像手机可以安装不同的 App，每个 App 做特定的事：
- 微信 → 聊天
- 地图 → 导航
- 相机 → 拍照

Claude Code 也可以为不同场景设置不同的"技能"。

---

## 常见的自定义技能

### 1. 学习助手技能

```
角色：学霸同学
特点：
- 用简单的解释复杂的概念
- 经常举生活例子
- 会出练习题巩固
- 语气鼓励积极

触发词："帮我学习"
```

### 2. 写作助手技能

```
角色：资深编辑
特点：
- 语法正确
- 逻辑清晰
- 结构合理
- 会给修改建议

触发词："帮我写作"
```

### 3. 代码导师技能

```
角色：耐心的编程老师
特点：
- 解释清晰
- 代码有注释
- 循序渐进
- 鼓励动手实践

触发词："教我写代码"
```

## 如何创建技能

### 步骤一：定义场景

```
我经常需要：[场景描述]
```

### 步骤二：设计角色

```
请用这个角色帮我：
- 身份：[身份]
- 专业背景：[背景]
- 说话风格：[风格]
- 特殊要求：[要求]
```

### 步骤三：设置触发词

```
当我说 "[触发词]" 时，请用这个技能帮我。
```

---

## 使用技能的示例

### 设置技能

```
请记住，从现在起，当我发"[学习]"开头的问题时，请以学霸同学的身份，用简单的语言、举生活例子帮我学习。

例如：
你：[学习] 什么是光合作用
我：好！让我用超简单的话解释...
```

### 使用技能

```
[学习] 帮我复习三角函数
[学习] 解释一下这个化学方程式
[写作] 帮我写一封道歉邮件
```

## 管理你的技能

### 查看当前技能

```
请问你现在记得哪些技能？
```

### 更新技能

```
请更新 [技能名]：
[新内容]
```

### 删除技能

```
请删除 [技能名]
```

## 小测验

**1. 自定义技能的核心是？**
- A. 让 AI 记住更多内容
- B. 为特定场景定制 AI 的角色和行为
- C. 让 AI 变得更聪明

**2. 触发词的作用是？**
- A. 让 AI 知道要用哪个技能
- B. 让 AI 更慢
- C. 没有作用

**3. 一个好的技能应该包含？**
- A. 角色、特点、触发词
- B. 越详细越好
- C. 只要有触发词就行

---

## 答案

1-B | 2-A | 3-A
