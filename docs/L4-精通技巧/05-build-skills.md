# 构建自己的技能

## 自定义技能是什么

为特定场景定制的 AI 配置。像手机 App：微信聊天、地图导航、相机拍照。

## 创建步骤

**1. 定义场景**：`我经常需要 [场景]`

**2. 设计角色**：
```
身份：[身份]
风格：[风格]
特殊要求：[要求]
```

**3. 设置触发词**：
```
当我说"[触发词]"时，用这个技能帮我。
```

## 示例

```
技能：学习助手
触发词：[学习]

当我说：
[学习] 帮我复习三角函数
[学习] 解释化学方程式

AI：以学霸同学身份，用简单语言、举生活例子帮我。
```

## 常见技能

**学生党**：
| 技能 | 触发词 | 用途 |
|------|--------|------|
| 学习辅导 | [学] | 解释概念、出题 |
| 作业帮手 | [作] | 检查作业 |
| 备考神器 | [考] | 复习计划 |

**职场人**：
| 技能 | 触发词 | 用途 |
|------|--------|------|
| 邮件助手 | [邮] | 写邮件 |
| 汇报模板 | [报] | 写报告 |
| 会议纪要 | [会] | 整理会议 |
