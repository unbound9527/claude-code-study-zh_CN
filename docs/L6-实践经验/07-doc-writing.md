# 文档写作

## 常用文档类型

| 类型 | 说明 |
|------|------|
| README | 项目说明、安装、使用 |
| API 文档 | 接口说明、参数、返回值 |
| 代码注释 | 函数说明、逻辑解释 |
| 技术方案 | 架构设计、实现细节 |
| 用户手册 | 功能说明、操作指南 |

## README 写作模板

```
# 项目名称

一句话描述项目。

## 功能特性
- 特性 1
- 特性 2

## 安装
```bash
[安装命令]
```

## 使用
[基本用法]

## 配置
[配置说明]
```

## API 文档模板

```
## 接口名称

**功能**：做什么

**请求**：
- Method: GET/POST
- URL: /api/xxx
- 参数：
  | 参数名 | 类型 | 必填 | 说明 |
  |-------|------|------|------|

**响应**：
```json
{
  "code": 0,
  "data": {}
}
```

**示例**：
```bash
curl ...
```
```

## 代码注释规范

| 内容 | 示例 |
|------|------|
| 函数说明 | `// 计算用户年龄，基于出生日期` |
| 参数 | `@param {string} name - 用户名` |
| 返回值 | `@returns {Promise<User>}` |
| 注意事项 | `// 注意：需要先登录` |

## 写作技巧

| 技巧 | 说明 |
|------|------|
| 先大纲后内容 | 先列提纲再写正文 |
| 读者视角 | 想象读者会问什么问题 |
| 例子驱动 | 重要概念配实际例子 |
| 图文并茂 | 复杂流程配合流程图 |
