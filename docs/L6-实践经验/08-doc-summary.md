# 文档总结

## 场景

| 场景 | 说明 |
|------|------|
| 长文摘要 | 论文、报告、文章提炼要点 |
| 会议记录 | 提取决议、待办、讨论点 |
| 书籍笔记 | 章节核心内容整理 |
| 代码文档 | 从代码生成文档说明 |

## 常用句式

```
请总结以下内容：
[粘贴内容]

要求：
- 提取 [N] 个核心要点
- 保留关键数据和结论
- 用 bullet points 呈现
- 标注重难点
```

## 会议记录总结

```
请整理会议记录，提取：
1. 讨论的主要问题
2. 作出的决定
3. 待办事项（负责人 + 截止时间）
4. 下次会议时间

格式：
## 决议
## 待办
## 下次会议
```

## 书籍/文章笔记

```
请为以下内容生成复习笔记：
[粘贴内容]

格式：
## 核心概念（3-5个）
## 重点解读
## 常见面试题
## 实践应用
```

## 要点提取技巧

| 技巧 | 说明 |
|------|------|
| 找关键词 | 重复出现的词往往是重点 |
| 看结论句 | 段落首尾通常是结论 |
| 提取数据 | 数字、百分比、排名等 |
| 保留术语 | 领域专有名词不能丢 |
