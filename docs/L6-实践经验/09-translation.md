# 翻译润色

## 中英互译

| 场景 | 说明 |
|------|------|
| 技术文档 | 保持术语准确、格式一致 |
| 学术论文 | 符合学术规范、用词正式 |
| 商业邮件 | 语气得体、专业 |
| 日常沟通 | 简洁、口语化 |

## 翻译常用句式

```
请翻译以下内容为 [目标语言]：

要求：
- 保持 [正式/口语化/学术] 语气
- 技术术语保留英文原词
- [其他要求]

[粘贴内容]
```

## 技术文档翻译

```
请翻译为英文技术文档：
- 保持技术术语准确
- 使用业界常用表达
- 代码注释保持英文
- 提供术语对照表

[粘贴中文技术文档]
```

## 英文润色

```
请润色以下英文：
[粘贴英文]

目标：
- 语法正确
- 表达地道
- 语气符合 [学术/商务/日常]
- 标出不确定的词汇
```

## 降重改写

在不改变意思的前提下改写，降低重复率：

```
请改写以下段落，降低重复率：
[粘贴内容]

要求：
- 保持原意
- 换用同义词
- 调整句式结构
- 不改变专业术语
```

## 术语处理

| 处理方式 | 说明 | 示例 |
|----------|------|------|
| 保留原词 | 通用技术术语 | AI、API、HTTP |
| 首次翻译 | 生僻术语首次出现时 | 深度学习(Deep Learning) |
| 创建对照表 | 正式文档 | 附术语表 |
