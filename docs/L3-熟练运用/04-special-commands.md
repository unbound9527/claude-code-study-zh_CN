# 特殊命令

## 常用命令

| 命令 | 作用 |
|------|------|
| `/help` | 显示帮助 |
| `/clear` | 清空当前对话（不删历史） |
| `/compact` | 压缩上下文 |
| `/undo` | 撤销上一条回答 |
| `/redo` | 重做 |

## 键盘快捷键

| 快捷键 | 作用 |
|--------|------|
| Enter | 发送 |
| Shift+Enter | 换行 |
| ↑ | 上一条输入 |
| Ctrl+C | 取消 |
