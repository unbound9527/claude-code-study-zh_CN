# 特殊命令

## 本章目标

- 学会使用 Claude Code 的快捷命令
- 知道 /help、/clear、/compact 等的作用

---

## 常用命令一览

| 命令 | 作用 |
|------|------|
| `/help` | 显示帮助信息 |
| `/clear` | 清除当前对话历史 |
| `/compact` | 压缩上下文 |
| `/undo` | 撤销上一个操作 |
| `/redo` | 重做 |

---

## /help - 帮助

输入：

```
/help
```

**作用：** 显示所有可用命令和使用说明

---

## /clear - 清空对话

输入：

```
/clear
```

**作用：** 清除当前对话的所有历史，开始全新的对话

**注意：**
- 不会删除之前的对话记录（可以在侧边栏找到）
- 只是清空当前对话的上下文

---

## /compact - 压缩上下文

输入：

```
/compact
```

**作用：** 当对话太长时，压缩历史记录，保留核心信息

**使用场景：**
- 对话超过 20 轮
- AI 开始"失忆"
- 需要继续一个长话题

---

## /undo - 撤销

输入：

```
/undo
```

**作用：** 撤销 AI 最后一次回答

**使用场景：**
- AI 回答错了
- 你不小心发送了
- 想换一种问法

---

## /redo - 重做

输入：

```
/redo
```

**作用：** 重新生成 AI 的回答（在你撤销之后）

---

## 键盘快捷键

| 快捷键 | 作用 |
|--------|------|
| Enter | 发送消息 |
| Shift + Enter | 换行（输入多行） |
| Ctrl + C | 取消当前操作 |
| ↑ | 重新输入上一条 |
| Tab | 自动补全（某些场景） |

---

## 跟做

2. 输入 `/clear`，确认清除对话
3. 问一个问题，然后试试 `/undo`

## 小测验

**1. /clear 会删除对话历史吗？**
- A. 会
- B. 不会，只是在当前对话清空
- C. 不确定

**2. 什么命令可以压缩上下文？**
- A. /help
- B. /clear
- C. /compact

**3. Shift + Enter 的作用是？**
- A. 发送消息
- B. 换行
- C. 清空输入框

---

## 答案

1-B | 2-C | 3-B
