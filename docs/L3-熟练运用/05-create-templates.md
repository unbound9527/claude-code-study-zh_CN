# 制作自己的模板

## 本章目标

- 理解什么是 Prompt 模板
- 能创建常用的 Prompt 模板
- 学会保存和使用自己的模板

---

## 什么是 Prompt 模板？

**模板 = 固定格式 + 变量**

就像填空题一样，模板让你不用每次重新写完整的 prompt。

---

## 模板的组成

```
[角色] + [任务] + [格式要求] + [变量]
```

| 部分 | 示例 |
|------|------|
| 角色 | "你是一个高中老师" |
| 任务 | "帮我解释 [主题]" |
| 格式 | "用 [字数] 字以内" |
| 变量 | [主题]、[字数] = 待填空 |

---

## 创建模板

### 模板一：学习解释模板

```
你是一个 [学科] 老师，帮我：

1. 用 [难度] 的语言解释 [主题]
2. 举 [数量] 个生活例子
3. 出 [数量] 道练习题

格式：用表格呈现
```

### 模板二：写作模板

```
帮我写 [类型]：

- 主题：[主题]
- 字数：[字数]
- 风格：[风格]
- 受众：[受众]

格式：[格式要求]
```

### 模板三：代码模板

```
用 [语言] 写 [功能]：

- 输入：[输入说明]
- 输出：[输出说明]
- 要求：[其他要求]

注释：详细注释关键步骤
```

---

## 保存模板的方法

### 方法一：复制粘贴到笔记

把常用的模板保存到：
- 备忘录
- 笔记软件（Notion、印象笔记等）
- 文档

### 方法二：创建快捷短语

在笔记里给每个模板起个名字：

```
【学习解释】→ 你是一个[学科]老师，帮我解释[主题]
【邮件模板】→ 帮写[类型]邮件，收件人[对象]
【代码模板】→ 用[语言]写[功能]，输入[说明]
```

## 使用模板

### 步骤一：打开模板

从笔记复制模板

### 步骤二：填入变量

```
【学习解释】
你是一个历史老师，帮我：

1. 解释"明治维新"
2. 举3个例子
3. 出3道练习题
```

## 常用模板库

### 1. 解释概念

```
你是一个[学科]老师。请用简单的话解释[概念]，
并举一个[例子数量]个生活中的例子。
```

### 2. 写邮件

```
请帮我写一封[类型]邮件：
- 收件人：[人]
- 目的：[目的]
- 语气：[语气]
- 字数：[字数]
```

### 3. 检查语法

```
请检查以下文字的语法和错别字：
[粘贴文字]

然后用修改后的版本回复我。
```

### 4. 总结笔记

```
请帮我总结以下内容：
[粘贴内容]

要求：
- 提取[数量]个核心要点
- 用 bullet points
- 标注重难点
```

## 小测验

**1. Prompt 模板的作用是？**
- A. 让 AI 更聪明
- B. 省去重复输入固定内容的麻烦
- C. 让回答更快

**2. 模板中的变量一般用什么表示？**
- A. [变量名]
- B. (变量名)
- C. <变量名>

**3. 模板应该保存在哪里？**
- A. 只在脑子里
- B. 笔记软件或文档，方便复制使用
- C. 不需要保存

---

## 答案

1-B | 2-A | 3-B
