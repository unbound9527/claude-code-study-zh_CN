# 制作自己的模板

## 模板是什么

**模板 = 固定格式 + 变量**，像填空题，不用每次重写。

## 模板结构

```
[角色] + [任务] + [格式要求] + [变量]
```

## 常用模板

**学习解释**：
```
你是一个[学科]老师，帮我：
1. 用[难度]的话解释[主题]
2. 举[数量]个例子
3. 出[数量]道练习题
格式：用表格
```

**写作**：
```
帮我写[类型]：
- 主题：[主题]
- 字数：[字数]
- 风格：[风格]
```

**代码**：
```
用[语言]写[功能]：
- 输入：[说明]
- 输出：[说明]
要求：[其他]
```

## 保存方式

保存到笔记软件（Notion、备忘录等），起简短名字方便调用。
