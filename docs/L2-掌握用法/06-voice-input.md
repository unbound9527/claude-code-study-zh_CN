# 语音输入详解

## 本章目标

- 掌握各种操作系统的语音输入方法
- 学会高效使用语音输入
- 能解决常见的语音输入问题

---

## 为什么用语音输入？

| 输入方式 | 速度 | 适合场景 |
|----------|------|----------|
| 打字 | 慢 | 精确输入、长文本 |
| 语音 | 快 | 大段文字、口语表达 |

**语音输入比打字快 3-5 倍！**

---

## Windows 语音输入

### 快捷键

| 功能 | 快捷键 |
|------|--------|
| 开始/停止听写 | `Win + H` |

### 使用步骤

1. 点击输入框
2. 按 `Win + H`
3. 开始说话
4. 说完后再次按 `Win + H` 停止

### 技巧

- 吐字清晰
- 语速适中
- 适当停顿

---

## macOS 语音输入

### 快捷键

| 功能 | 快捷键 |
|------|--------|
| 开始/停止听写 | `Fn + .` 或 `Ctrl + Cmd + .` |

### 使用步骤

1. 点击输入框
2. 按 `Fn + .`
3. 开始说话
4. 说完后再次按相同快捷键

### 语音命令

| 命令 | 效果 |
|------|------|
| "新段落" | 换行 |
| "句号" | 添加句号 |
| "逗号" | 添加逗号 |
| "删除" | 删除上一句 |
| "重做" | 重新输入 |

## 提高识别准确率

### ✅ 好的做法

- 吐字清晰
- 语速适中（不要太快）
- 背景安静
- 简短停顿分隔句子

### ❌ 避免

- 同时说话和听音乐
- 在嘈杂环境
- 说话太快或含糊
- 中英文混杂（除非必要）

---

## 常见问题解决

### 问题一：识别不准

**解决方法**：
- 说得慢一点
- 确保麦克风清楚
- 检查网络连接

### 问题二：无法唤醒

**解决方法**：
- Windows：检查"语音设置"是否启用
- macOS：检查"辅助功能"中的语音权限

### 问题三：标点错乱

**解决方法**：
- 手动添加标点符号
- 使用"标点符号名称"（如"句号"、"逗号"）
