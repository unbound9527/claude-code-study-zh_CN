# 语音输入详解

## 快捷键

| 系统 | 快捷键 |
|------|--------|
| Windows | `Win + H` |
| macOS | `Fn + .` 或 `Ctrl + Cmd + .` |

## 技巧

| 技巧 | 说明 |
|------|------|
| 清晰吐字 | 语速适中 |
| 添加标点 | 说"逗号""句号" |
| 编辑 | "删除""重做" |

## 提高准确率

| ✅ 这样做 | ❌ 避免 |
|----------|--------|
| 吐字清晰 | 同时听音乐 |
| 背景安静 | 嘈杂环境 |
| 简短停顿 | 说话太快 |
