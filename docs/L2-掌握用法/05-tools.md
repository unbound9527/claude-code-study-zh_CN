# Claude Code 能做什么：工具详解

## 本章目标

- 理解 Claude Code 的内置工具
- 知道什么时候 AI 会使用什么工具
- 能主动让 AI 使用特定工具

---

## Claude Code 不只是聊天

当你和 Claude Code 对话时，它背后可以做很多事：

```
你说："帮我读取这个文件"
    ↓
Claude Code 使用 [Read] 工具
    ↓
读取文件内容
    ↓
分析后回复你
```

**工具 = Claude Code 的"手"**，让它不只能说话，还能做事。

---

## 内置工具一览

| 工具 | 作用 | 例子 |
|------|------|------|
| **Read** | 读取文件内容 | "看一下 src/app.js" |
| **Write** | 创建/写入文件 | "帮我创建 index.html" |
| **Edit** | 修改文件内容 | "改第三行的文字" |
| **Bash** | 执行命令 | "运行 npm install" |
| **Glob** | 搜索文件 | "找到所有 .js 文件" |
| **Grep** | 搜索内容 | "找到包含 'login' 的文件" |
| **WebFetch** | 获取网页 | "看一下这个网页内容" |
| **Agent** | 启动子代理 | "同时处理这三件事" |

---

## Read - 读取文件

### 什么时候用？

- 想让 AI 看某个文件的内容
- 想让 AI 分析代码
- 想让 AI 检查文档

### 示例

```
"请看一下 src/utils.js 这个文件"
"帮我读取这份文档"
"看看 App.css 的第 10-20 行"
```

### AI 的反应

当你让 AI 读取文件时，它会：
1. 使用 Read 工具读取
2. 分析内容
3. 给你反馈

---

## Write - 写入文件

### 什么时候用？

- 让 AI 创建新文件
- 让 AI 保存回答到文件

### 示例

```
"帮我创建一个 index.html 文件"
"把这段代码保存到 src/app.js"
```

### AI 的反应

```
你：请帮我创建一个 HTML 文件
AI：正在创建...
[Write] → 创建了 index.html
完成！我帮你创建了一个基础的 HTML 文件。
```

---

## Edit - 修改文件

### 什么时候用？

- 修改文件的部分内容
- 修复 bug
- 调整代码

### 示例

```
"把第三行的 'hello' 改成 'world'"
"在这个函数里加上错误处理"
"删除第五段的重复内容"
```

### AI 的反应

AI 会精确修改，不会覆盖整个文件。

---

## Bash - 执行命令

### 什么时候用？

- 运行代码
- 安装依赖
- 执行脚本

### 示例

```
"运行 npm install"
"执行 python main.py"
"帮我运行测试"
```

### 安全提示

某些命令需要授权才能执行，比如：
- 删除文件 (`rm -rf`)
- Git 推送 (`git push`)
- 系统命令

AI 会询问你是否允许。

---

## Glob - 搜索文件

### 什么时候用？

- 找特定类型的文件
- 不知道文件在哪
- 批量找文件

### 示例

```
"找到项目里所有的 .js 文件"
"找出 src 目录下所有的组件"
"列出所有测试文件"
```

---

## Grep - 搜索内容

### 什么时候用？

- 在文件中找特定文字
- 找某个函数在哪里定义
- 找某个变量被使用的地方

### 示例

```
"在 src 目录下找到 'login' 相关的代码"
"找出所有用到 'useState' 的地方"
"查找包含 'TODO' 的文件"
```

---

## WebFetch - 获取网页

### 什么时候用？

- 获取网页内容
- 分析网页信息
- 抓取数据

### 示例

```
"帮我看看 https://example.com 的内容"
"获取这个网页的主要信息"
```

---

## Agent - 启动子代理

### 什么时候用？

- 同时处理多个任务
- 需要并行工作

### 示例

```
"请同时帮我：1. 写代码 2. 写测试 3. 写文档"
```

---

## 主动指定工具

### 用法

```
"请用 [工具名] [任务]"

示例：
"请用 Read 工具看一下这个文件"
"用 Bash 运行一下这个脚本"
```

### 不指定时的默认行为

AI 会根据你的需求自动选择合适的工具。
