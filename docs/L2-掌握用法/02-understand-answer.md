# 理解 AI 的回答

## 本章目标

- 学会判断 AI 回答的质量
- 识别回答中的有用信息和局限性
- 知道什么时候需要追问

---

## AI 的回答不都一样

AI 的回答可能有：

| 情况 | 说明 |
|------|------|
| 正确且完整 | 这是我们想要的 |
| 正确但不完整 | 需要追问 |
| 部分错误 | 需要纠正 |
| 完全错误 | 需要重新问 |

---

## 如何判断回答质量

### 1. 检查事实

AI 可能会"编造"看似真实的信息，叫 **"幻觉"**（hallucination）。

**检验方法：**
- 用搜索引擎核实关键信息
- 问 AI "请提供这个信息的来源"

### 2. 检查逻辑

- 推理过程是否通顺？
- 结论是否从前提推导出来？

### 3. 检查完整性

- 回答了所有问题吗？
- 例子足够说明观点吗？

---

## 回答的结构

好的回答通常包含：

```
┌─────────────────────────────────┐
│  直接答案（核心回答）            │
├─────────────────────────────────┤
│  解释说明（为什么/怎么做）       │
├─────────────────────────────────┤
│  例子（举一个或多个）           │
├─────────────────────────────────┤
│  延伸（你可以继续问）            │
└─────────────────────────────────┘
```

---

## 什么时候需要追问？

| 情况 | 你可以问 |
|------|----------|
| 回答太笼统 | "请详细说明" |
| 举例不够 | "再举一个例子" |
| 有术语不懂 | "这个词是什么意思" |
| 结论太快 | "这个结论怎么得出的" |
| 想要替代方案 | "还有其他方法吗" |

---

## 追问技巧

### 技巧一：指定方向

```
❌ 不好：追问 "为什么？"

✅ 好：
- "第三步的逻辑是什么？"
- "能解释一下第二点吗？"
```

### 技巧二：要求简化

```
❌ 不好："再详细点"

✅ 好：
- "用小学生能懂的话解释"
- "用一句话总结"
- "给个最简单的版本"
```

### 技巧三：要求扩展

```
❌ 不好："多点内容"

✅ 好：
- "再给 3 个例子"
- "从行业角度再分析一下"
```

## 识别 AI 的局限性

### AI 原本不知道的（有了工具后可以做到）

| 内容 | 过去为什么不知道 | 现在如何获取 |
|------|------------|----------|
| 今天发生的事 | 训练数据有截止日期 | 通过 WebFetch 访问网页获取 |
| 你的个人信息 | 没有你的背景信息 | 通过 Memory 功能记住，或在对话中告诉它 |
| 实时数据 | 不能上网查询 | 通过 MCP 工具或 WebFetch 获取 |
| 文件内容 | 不知道项目里有什么 | 通过 Read/Glob/Grep 工具读取 |

**提示**：AI 的"不知道"大多是指它**没有主动获取**的能力，而不是它**完全无法**知道。只要给它合适的工具或上下文，它就能完成很多之前"做不到"的事。

### AI 会犯的错

| 错误类型 | 示例 | 怎么办 |
|----------|------|--------|
| 编造事实 | "这句话是马克·吐温说的"（其实不是） | 核实 |
| 推理错误 | 数学推导出错 | 验算 |
| 偏见 | 对某些群体有刻板印象 | 保持警惕 |

## 跟着做

1. 问 AI：`什么是量子计算？`

2. 检查回答：
   - 核心定义清楚吗？
   - 有例子吗？
   - 有延伸阅读建议吗？

3. 追问：`能举个生活中的例子吗？`

4. 再追问：`适合给小学生讲吗？怎么讲？`

---

## 练一练

1. 问 AI 一个你不熟悉的话题
2. 仔细阅读回答
3. 找出你不理解的部分
4. 追问直到理解

---

## 小测验

**1. AI 幻觉是指？**
- A. AI 会做梦
- B. AI 编造看似真实的信息
- C. AI 看到了不存在的东西

**2. 追问"再详细点"的问题是什么？**
- A. 太模糊，不知道要详细什么
- B. 正确的方式
- C. 会让 AI 生气

**3. AI 不知道什么？**
- A. 历史上的事实
- B. 今天的天气
- C. 编程知识

---

## 答案

1-B | 2-A | 3-B
