# 它答错了怎么办

## 本章目标

- 识别 AI 什么时候答错了
- 掌握纠正和追问的技巧
- 能引导 AI 给出正确答案

---

## AI 答错的常见情况

| 情况 | 示例 | 你应该 |
|------|------|--------|
| 答非所问 | 问 A 它答 B | 重新明确问题 |
| 信息错误 | 把张冠李戴 | 指出错误并要求核实 |
| 不完整 | 只答了一半 | 追问剩下的 |
| 太简单 | 给了废话 | 要求深入 |
| 太复杂 | 专业术语太多 | 要求简化 |

---

## 纠正技巧

### 技巧一：直接指出错误

```
❌ 不好：
"你好像搞错了"

✅ 好：
"第三点不对。爱因斯坦的出生年份是 1879 年，不是 1880 年。请更正。"
```

### 技巧二：提供正确信息

```
❌ 不好：
"你说得不对"

✅ 好：
"实际上我的手機是 iPhone 15，不是三星。请重新推荐配件。"
```

### 技巧三：解释你要什么

```
❌ 不好：
"不是我想要的"

✅ 好：
"我要的不是这个方向。我要的是适合大学生的平价穿搭，不是职场穿搭。请重新写。"
```

---

## 追问技巧

### 1. 指定方向追问

```
❌ 不好：
"不对"

✅ 好：
"方向不对。我要的是 [具体说明]。请重新考虑 [另一个角度]"
```

### 2. 要求简化/深入

```
"太简单了，能再深入一点吗？"
"太专业了，能用大白话解释吗？"
"例子不够，能再举 3 个吗？"
```

### 3. 要求换一种方式

```
"能换个角度吗？"
"能用表格形式吗？"
"能先总结再展开吗？"
```

---

## 实用句式

### 要求重新回答

```
请重新回答。
```

### 要求更简洁

```
太长了。请精简到 100 字以内。
```

### 要求更详细

```
太笼统了。请详细说明 [某个点]。
```

### 要求换角度

```
这个角度不好。请从 [另一个角度] 重新分析。
```

### 指出事实错误

```
[某个具体事实] 是不对的，应该是 [正确内容]。请更正。
```

### 调整风格

```
太正式了。能用更口语化的方式表达吗？"
```

---

## 迭代优化

有时候 AI 不会一次就答对，需要多轮调整：

```
第一轮：你 → AI
        ↓ 不满意
第二轮：你（指出问题）→ AI
        ↓ 还不满意
第三轮：你（继续调整）→ AI
        ↓ 满意
最终答案
```

**这是正常的！不要灰心。**

---

## 跟着做

1. 问 AI：
```
请解释什么是"内卷"
```

2. 如果回答太学术化，追问：
```
太学术了。请用高中生能听懂的话，举生活例子解释。
```

3. 如果例子不够，追问：
```
再举一个你没说过的例子
```

4. 如果你满意了，说：
```
很好，就这个方向。能再补充 [其他方面] 吗？
```

---

## 练一练

1. 让 AI 给你推荐一本书（说清楚类型）
2. 如果推荐不满意，尝试纠正和追问
3. 直到得到满意的推荐

## 小测验

**1. AI 答错了，你不应该？**
- A. 指出具体错误
- B. 提供正确信息
- C. 直接骂它

**2. "太长了，请精简到 50 字"是什么技巧？**
- A. 指出错误
- B. 要求调整
- C. 换角度

**3. AI 需要多轮调整是？**
- A. 不正常的
- B. 正常的，迭代优化是正常的
- C. 说明 AI 很笨

---

## 答案

1-C | 2-B | 3-B
