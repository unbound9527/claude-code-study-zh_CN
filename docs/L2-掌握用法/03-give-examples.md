# 给例子让它更懂你

## 为什么例子重要

例子帮助 AI 理解：风格、场景、期望格式。

## 给例子的方式

**1. 直接给输入输出示例**
```
按这个格式写标题：
输入：某公司发布新产品
输出：标题1 | 标题2 | 标题3

现在帮我写：大学食堂推出健康餐
```

**2. 给风格示例**
```
按这个风格写：
"天啊！绝绝子！超好吃！"

帮我推荐这款耳机
```

**3. 给类比**
```
类比：电流像水流，电压像水压

用同样方式解释热传导
```

## 有效 vs 无效例子

| 有效 | 无效 |
|------|------|
| 相关、具体、有输入输出 | 太泛、不相关、模糊 |
