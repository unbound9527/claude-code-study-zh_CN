# 给例子让它更懂你

## 本章目标

- 理解"例子"的重要性
- 学会用类比和示例来表达
- 能给出有效的例子

---

## 为什么例子很重要？

有一句话叫：**"举例子比讲道理更有效"**

例子帮助 AI 理解：
- 你想要什么风格
- 具体是什么场景
- 期望的输出格式

---

## 给例子的方式

### 方式一：直接给输入输出示例

```
请按照这个格式写新闻标题：

示例：
输入：某公司发布新产品
输出：标题 1 | 标题 2 | 标题 3

现在帮我为这个内容写标题：
输入：大学食堂推出健康餐
```

### 方式二：给风格示例

```
请按照这个风格写文案：

风格参考：李佳琦直播带货风格
- 夸张感叹
- 突出卖点
- 引导购买

现在帮我推荐这款耳机
```

### 方式三：给类比

```
请用这个类比来解释：

类比：电流就像水流，电压就像水压，电阻就像管道粗糙度

请用同样的方式解释：热传导
```

---

## 有效例子的特征

### ✅ 好的例子

| 特征 | 示例 |
|------|------|
| 相关 | 和你的问题同一领域 |
| 具体 | 不是泛泛而谈 |
| 清晰 | 容易理解 |
| 有输入输出 | 包含完整对话 |

### ❌ 不好的例子

| 问题 | 示例 |
|------|------|
| 太泛 | "就像一般那样" |
| 不相关 | 用做菜的例子解释编程 |
| 模糊 | "写得差不多就行" |

## 实战练习

### 练习一：指定文案风格

**你想写的：**
- 一条朋友圈推荐你开的奶茶店

**给例子：**
```
帮我写朋友圈文案，推荐我新开的奶茶店

风格参考（模仿这个）：
"天啊！终于找到了！这款奶茶真的绝绝子！奶盖超厚，茶味很正，一口下去整个人都升华了！

现在帮我写：产品是芋泥波波奶茶，18 元一杯，位置在大学城
```

### 练习二：指定格式

**你想写的：**
- 给老师的请假条

**给例子：**
```
帮我写请假条

格式参考：
[标题]
[称呼]：
[正文，说明原因和时间]
[署名]
[日期]

现在帮我写：请假 3 天，因为感冒
```

### 练习三：指定语气

**你想写的：**
- 安慰朋友的生日祝福

**给例子：**
```
安慰朋友，TA 考试没考好

语气参考（温柔鼓励型）：
"我知道你很难过，这次没考好不代表什么。你一直都很努力，下次一定会更好的！

现在帮写：朋友小明这次数学考了 70 分（平时 90+）
```

## 跟着做

1. 打开 Claude Code
2. 输入：

```
请用这种风格回复消息：

风格参考（朋友间的轻松聊天）：
"哈哈哈哈哈笑死" "我懂你" "冲冲冲"

现在帮我想一个回复：朋友说他刚买了一个超好用的键盘
```

3. 看看 AI 的回复是否像朋友聊天

## 练一练

选择一个场景，给 AI 一个例子，然后让它完成：

1. **邮件** - 写一封给老板的工作汇报
2. **评论** - 给这部电影写一句评价
3. **介绍** - 介绍你的专业给高中生

## 小测验

**1. 给例子的主要目的是？**
- A. 让 AI 多读字
- B. 帮助 AI 理解你想要什么
- C. 没有实际作用

**2. 以下哪个是好的例子？**
- A. "写得好看点"
- B. "像一般那样写"
- C. "参考这篇范文的结构和语气" + 粘贴范文

**3. 给格式例子时，最好包含？**
- A. 只有标题
- B. 完整的输入和输出示例
- C. 只要口头描述

---

## 答案

1-B | 2-C | 3-B
