# Prompt 高级技巧

## 本章目标

- 掌握高级 Prompt 编写技巧
- 能针对不同场景优化 Prompt
- 能调试和迭代 Prompt

---

## 基础回顾

### Prompt 公式

```
角色 + 任务 + 要求 + 示例（可选）
```

---

## 技巧一：角色扮演

### 基础用法

```
你是一个 [职业/身份]，请帮我 [任务]
```

### 进阶用法

```
你扮演一个 [角色]，有 [年数] 年经验，
专长于 [领域]，请用 [风格] 帮我 [任务]
```

**示例**：
```
你扮演一个有 10 年经验的高级产品经理，
专长于电商领域，用苏杰的《人人都是产品经理》风格，
帮我分析这个需求文档。
```

---

## 技巧二：结构化输出

### 指定格式

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| 表格 | 行列对齐 | 对比分析 |
| JSON | 程序化 | 代码集成 |
| Markdown | 文档化 | 报告文档 |
| bullet points | 简洁列表 | 快速概览 |

### 示例

**表格格式**：
```
请用表格对比 A 和 B，包含：维度、A、B、结论
```

**JSON 格式**：
```
请用 JSON 格式返回：
{
  "name": string,
  "age": number,
  "skills": string[]
}
```

**Markdown 格式**：
```
请用以下格式组织：
## 标题
### 小标题
- 要点 1
- 要点 2
```

---

## 技巧三：思维链提示

让 AI 逐步思考，而不是直接给答案。

### 用法

```
请逐步分析这个问题：
1. 首先理解问题
2. 其次分析关键点
3. 最后给出结论
```

### 进阶

```
你的思维过程应该是：
[定义问题] → [分析原因] → [评估选项] → [给出建议]
```

---

## 技巧四：few-shot 学习

通过例子让 AI 理解你要的模式。

### 格式

```
示例 1：
输入：[输入1]
输出：[输出1]

示例 2：
输入：[输入2]
输出：[输出2]

现在请处理：
输入：[新输入]
```

### 示例

```
示例：
输入：今天很开心
输出：🎉 今天很开心！

输入：考试没考好
输出：😢 考试没考好...

现在请处理：
输入：收到了心仪的 offer
```

## 技巧五：限制和约束

### 明确限制

| 类型 | 示例 |
|------|------|
| 长度 | "不超过 100 字" |
| 数量 | "列出 5 个" |
| 格式 | "只用中文" |
| 风格 | "口语化" |

### 示例

```
请帮我写一段话，要求：
- 长度：50-80 字
- 语气：轻松友好
- 必须包含：emoji
- 不能包含：专业术语
```

---

## 技巧六：多步骤分解

### 主动分解

```
请帮我完成 [大任务]，分三步：

第一步：[子任务 1]
第二步：[子任务 2]
第三步：[子任务 3]
```

### 让 AI 分解

```
[大任务]，请先告诉我需要哪些步骤。
```

---

## 技巧七：引用和参考

### 指定参考

```
参考以下内容回答：
[粘贴参考内容]

问题：[你的问题]
```

### 指定来源

```
请基于最新的研究回答，不要编造数据。
```

---

## 技巧八：迭代优化

### 初始 Prompt

```
帮我写一段产品介绍
```

### 第一次优化

```
帮我写一段 [产品名称] 的产品介绍，
目标用户是 [用户群]，
突出 [卖点]，
字数 100 字以内。
```

### 第二次优化

```
帮我写一段 [产品名称] 的产品介绍，
目标用户是 [用户群]，
突出 [卖点]，
字数 100 字以内，
风格参考 [参考风格或链接]。
```

## 常见错误

### 错误一：过于笼统

❌：
```
帮我看看
```

✅：
```
请检查这段代码的安全问题：
[粘贴代码]
```

### 错误二：要求矛盾

❌：
```
写一段很长但很简短的内容
```

✅：
```
用 3 句话解释概念
```

### 错误三：缺少上下文

❌：
```
这个方案怎么样？
```

✅：
```
请评估这个营销方案：
- 预算：10 万
- 目标：拉新 1000 用户
- 渠道：短视频 + 公众号
[粘贴方案]
```

## 本章小结

**L2 掌握用法**让你的 AI 使用从"能用"升级到"用得好"：

| 核心技能 | 关键要点 |
|---------|---------|
| 清晰表达 | 角色 + 任务 + 要求 + 示例，结构化你的需求 |
| 工具使用 | 工具是 AI 的"手脚"，让 AI 能读文件、写代码、执行命令 |
| 理解回答 | 会判断质量、会追问、会识别 AI 的局限性 |
| 迭代优化 | 先粗糙后精细，通过多轮对话逐步达到目标 |

**高级技巧精华**：
```
✓ 角色扮演 → 指定 AI 的专业身份
✓ 结构化输出 → 指定 JSON/表格/Markdown 格式
✓ Few-shot → 给例子让 AI 理解你的模式
✓ 思维链 → 让 AI 逐步推理而非直接给答案
```

**本章重点记住**：好问题 + 好工具 = 好结果。AI 的能力取决于你如何表达需求。
