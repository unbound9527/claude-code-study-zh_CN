# 清晰表达需求

## 本章目标

- 理解为什么清晰表达很重要
- 学会使用"角色 + 任务 + 要求"公式
- 能写出清晰的 prompt

---

## 为什么同样问题效果不同？

你有没有遇到过：
- 问 AI 一个问题，它答非所问
- 明明是一个问题，AI 理解得完全不对

**原因**：你的 prompt 太模糊了。

---

## 清晰表达的公式

```
角色：你希望 AI 扮演什么？（可选）
任务：具体要做什么？
要求：有什么限制或特殊要求？
```

### 例子

**模糊版：**
```
帮我写一封信
```

**清晰版：**
```
角色：你是一个专业的商务秘书
任务：帮写一封给客户的道歉信
要求：
- 语气诚恳但不卑微
- 200 字以内
- 解释延迟原因是供应商问题
- 承诺 3 天内解决
```

---

## 公式拆解

### 1. 角色（可选）

告诉 AI 它的身份：

| 角色 | 示例 prompt |
|------|-------------|
| 老师 | "你是高中数学老师，帮我解释这个概念" |
| 翻译 | "你是一个专业翻译，帮我中译英" |
| 编辑 | "你是一个资深编辑，帮修改这篇文章" |
| 同学 | "你是一个学霸，帮我复习这个章节" |

### 2. 任务

具体说明要做什么：

❌ 模糊：
```
写代码
```

✅ 具体：
```
用 Python 写一个函数，接收一个列表，返回最大值和最小值
```

### 3. 要求

添加限制条件：

| 类型 | 示例 |
|------|------|
| 长度 | "100 字以内"、"不超过 3 段" |
| 格式 | "用表格"、"用 bullet points" |
| 风格 | "口语化"、"正式"、"轻松" |
| 语气 | "友善"、"严肃"、"幽默" |
| 受众 | "给小学生看"、"给专业人士看" |

### 练习一：写邮件

## 常见错误

### 错误一：假设 AI 知道你的情况

❌：
```
帮我看看这个方案
```

✅：
```
帮分析这个营销方案，预算 5 万，目标人群是大学生，产品是零食
```

### 错误二：要求不明确

❌：
```
写好一点
```

✅：
```
润色这篇文章，语言更生动，逻辑更清晰，不少于 500 字
```

### 错误三：一次要求太多

❌：
```
帮我写一篇论文，包括摘要、引言、方法、结论、参考文献，还要查重

## 小测验

**1. prompt 公式是什么？**
- A. 问题 + 答案
- B. 角色 + 任务 + 要求
- C. 开头 + 中间 + 结尾

**2. 以下哪个最清晰？**
- A. "帮我写信"
- B. "帮写一封请假邮件，100字以内，语气真诚"
- C. "写东西"

**3. "要求"部分可以不包括？**
- A. 长度限制
- B. 格式要求
- C. AI 的家庭住址

---

## 答案

1-B | 2-B | 3-C
