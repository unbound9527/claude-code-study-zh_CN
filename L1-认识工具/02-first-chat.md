# 第一次对话

## 打开 Claude Code

- **网页版**：浏览器打开 claude.ai/code，登录
- **桌面应用**：下载安装，登录

## 界面

```
┌─────────────────────────────────┐
│ [对话历史列表]                    │
├─────────────────────────────────┤
│ AI：你有什么需要帮忙的吗？          │
│                                 │
│ 你：输入问题...             [发送]│
└─────────────────────────────────┘
```

- 对话区：显示历史
- 输入框：输入问题
- Enter 或点击按钮发送

## 第一次对话

1. 输入问题
2. 按 Enter 或点击发送
3. 等待回复

AI 思考后显示回答，可能包含：直接答案、解释、例子、追问。

## 语音输入

| 系统 | 快捷键 |
|------|--------|
| Windows | Win + H |
| macOS | Fn + . 或麦克风图标 |

**技巧**：吐字清晰，可说"逗号""句号"添加标点。

## 对话管理

- **新建对话**：点击 New Chat，不删除历史
- **关闭**：直接关标签页或窗口
- **退出登录**：设置中操作
