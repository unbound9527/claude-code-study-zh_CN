# 认识 AI 与有效对话

## AI 是什么

AI（人工智能）不是真的"智能"，更像：

- 记忆力超强的图书馆管理员
- 读过全世界文章的复读机

它根据学过的内容组织回答，不是有真正的理解。

## Claude Code 是什么

Anthropic 开发的 AI 对话工具，通过工具调用可以"感知世界、执行动作"。

| 能力 | 说明 |
|------|------|
| 回答问题 | 知识问答、解释概念 |
| 编程 | 写代码、调试、分析 |
| 写作 | 写邮件、文章、代码 |
| 分析 | 数据、文本、代码分析 |

**有了工具后能做的事**：

| 原本不擅长 | 解决方案 |
|-----------|---------|
| 实时信息 | WebFetch 获取 |
| 精确计算 | Bash/计算器 |
| 记住信息 | Memory 功能 |
| 找文件 | Glob/Grep |

## 对话 vs 聊天

| 聊天 | Claude Code |
|------|-------------|
| 朋友记得上次说的 | 默认只记当前对话 |
| 朋友会猜你意思 | AI 只能按你说的理解 |
| 朋友会纠正你 | AI 尽量回答你 |

## 好问题 vs 坏问题

| 坏问题 | 好问题 | 为什么 |
|--------|--------|--------|
| "帮我看看" | "检查这段文字语法" | 太模糊 |
| "怎么做" | "做番茄钟 APP，告诉我步骤" | 缺背景 |
| "那个事" | "上周写的邮件再发一遍" | 指代不清 |

## 提问三法则

**1. 说清楚要什么**
```
❌ 帮我写东西
✅ 100字内写朋友圈文案，推荐新耳机
```

**2. 给背景**
```
❌ 这个方案怎么样
✅ 大学生校园卖零食，定价8元/包进价5元，合理吗
```

**3. 指定格式**
```
❌ 列几个学习方法
✅ 5个高效学习法，用表格，含方法名/步骤/适用场景
```

## 多窗口管理

不同项目、不同话题用不同对话。点击 New Chat 新建对话，悬停可改名或删除。
