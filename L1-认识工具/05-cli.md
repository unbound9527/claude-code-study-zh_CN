# CLI 命令行入门

## CLI 是什么

**CLI = Command Line Interface**，用文字命令操作电脑。Claude Code 运行在终端里。

## CLI vs GUI

| | CLI | GUI |
|--|-----|-----|
| 操作 | 打字 | 鼠标点 |
| 速度 | 快（熟练后） | 慢 |
| 自动化 | 容易 | 难 |

## 打开终端

| 系统 | 方法 |
|------|------|
| Windows | Win + R → 输入 `cmd` |
| Mac | Command + Space → 搜索 Terminal |
| Linux | Ctrl + Alt + T |

## 基础命令

**Windows：**
```bash
dir              # 列文件
cd 文件夹名       # 切换目录
mkdir 文件夹名     # 创建文件夹
del 文件名        # 删除文件
```

**Mac/Linux：**
```bash
ls               # 列文件
cd 文件夹名       # 切换目录
mkdir 文件夹名    # 创建文件夹
rm 文件名        # 删除文件
```
