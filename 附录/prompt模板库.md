# Prompt 模板库

这里收录了常用的 Prompt 模板，可以直接复制使用。

---

## 学习类

### 解释概念

```
你是一个 [学科] 老师。请用 [难度] 的话解释 [概念]：
- 难度：小白 / 高中生 / 大学生
- 举 [数量] 个生活例子
- 最后用一句话总结
```

### 出练习题

```
根据以下内容，出 [数量] 道 [题型]：
- 题型：选择题 / 填空题 / 问答题
- 附答案

[粘贴内容]
```

### 总结笔记

```
请整理以下笔记：
1. 提取 [数量] 个核心知识点
2. 标注重难点
3. 制作简易思维导图（用文字缩进表示）

[粘贴笔记]
```

## 写作类

### 写邮件

```
请帮我写一封 [类型] 邮件：
- 收件人：[对象]
- 目的：[目的]
- 语气：[语气]
- 字数：[字数]

[其他要求]
```

### 润色修改

```
请润色以下文字：
- 目标：更流畅 / 更简洁 / 更专业 / 更口语化
- 字数限制：[字数]

[粘贴原文]
```

### 写文案

```
帮我想 [数量] 个 [类型] 文案：
- 类型：朋友圈 / 小红书 / 微博 / 抖音
- 主题：[主题]
- 风格：[风格]
- 每个不超过 [字数] 字
```

## 工作类

### 写报告

```
请帮我写一份 [类型] 报告：
- 格式：[格式]
- 包含：[内容要求]
- 字数：[字数]

[背景信息]
```

### 会议纪要

```
请整理以下会议记录，输出格式：

## 会议基本信息
## 讨论要点
## 决议事项
## 待办事项（负责人 + 截止）
## 下次会议

[粘贴记录]
```

### 工作周报

```
请根据以下内容，生成 [日报/周报/月报]：

[粘贴原始信息]

格式：
## 完成事项
## 进行中
## 下周计划
## 需要支持
```

## 代码类

### 写代码

```
用 [语言] 写 [功能]：

- 输入：[说明]
- 输出：[说明]
- 要求：[其他]
- 注释：详细注释关键步骤
```

### 解释代码

```
请解释以下代码：
- 整体功能是什么
- 关键步骤的解释
- 可能的优化建议

[粘贴代码]
```

### Debug

```
我的代码报错了：
- 错误信息：[粘贴]
- 预期行为：[说明]
- 实际行为：[说明]

请帮我找出原因和修复方法。

[粘贴代码]
```

## 生活类

### 旅行规划

```
帮规划 [目的地] [天数] 旅行：
- 人数：[数量]
- 预算：[金额]
- 出行人群：[描述]
- 必去景点：[列表]

请给出每日行程、预算分配、注意事项。
```

### 选购建议

```
请比较 [产品 A] 和 [产品 B]：

| 对比项 | 产品 A | 产品 B |
|--------|--------|--------|
| 价格 | | |
| 功能 | | |
| 优缺点 | | |

我的需求是 [需求]，请给出推荐。
```

## 工具类

### 翻译

```
请翻译为 [目标语言]：
- 风格：[正式 / 口语 / 学术]
- 不直译，要符合目标语言习惯

[粘贴内容]
```

### 检查语法

```
请检查以下文字的语法和错别字，然后用修改后的版本回复我。

[粘贴内容]
```

---

## 使用方法

1. 复制模板
2. 把 [xxx] 换成你的具体内容
3. 根据需要删减或添加要求
4. 发送给 Claude Code
