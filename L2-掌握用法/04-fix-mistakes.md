# 它答错了怎么办

## 常见错误情况

| 情况 | 怎么办 |
|------|--------|
| 答非所问 | 重新明确问题 |
| 信息错误 | 指出并要求核实 |
| 不完整 | 追问剩下的 |
| 太简单/太复杂 | 要求调整 |

## 纠正技巧

**直接指出**：
```
❌ "你好像搞错了"
✅ "第三点不对，爱因斯坦出生于1879年，请更正"
```

**提供正确信息**：
```
✅ "实际是 iPhone 15，不是三星"
```

**说明你要什么**：
```
✅ "我要的是大学生平价穿搭，不是职场穿搭"
```

## 实用句式

| 目的 | 说法 |
|------|------|
| 精简 | "精简到100字" |
| 详细 | "详细说明XX点" |
| 换角度 | "从XX角度重新分析" |
| 简化 | "用大白话" |
| 换格式 | "用表格" |

## 迭代优化

多轮调整是正常的：
```
你 → AI（不满意）
你（指出问题）→ AI（还不满意）
你（继续调整）→ AI（满意）
```
