# Prompt 高级技巧

## 核心公式

```
角色 + 任务 + 要求 + 示例
```

## 技巧

### 1. 角色扮演
```
你是一个有10年经验的产品经理，专长电商领域
```

### 2. 结构化输出
| 格式 | 适用场景 |
|------|---------|
| 表格 | 对比分析 |
| JSON | 程序集成 |
| Markdown | 报告文档 |

### 3. 思维链
```
请逐步分析：先理解问题 → 分析关键点 → 给出结论
```

### 4. Few-shot
```
输入：今天很开心
输出：🎉 今天很开心！

输入：考试没考好
现在请处理：收到了offer
```

### 5. 限制约束
- 长度："不超过100字"
- 数量："列出5个"
- 风格："口语化"

### 6. 多步骤分解
```
帮我完成大任务，分三步：
第一步... 第二步... 第三步...
```

## 常见错误

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| "帮我看看" | "检查这段代码安全问题" |
| "写长一点但要简短" | "3句话解释" |
| "这个方案怎么样" | "评估营销方案：预算10万，目标..." |
