# 清晰表达需求

## 问题出在哪

同样问题效果不同，原因是 **prompt 太模糊**。

## 核心公式

```
角色 + 任务 + 要求 + 示例
```

## 公式详解

**1. 角色**（可选）
- 帮助 AI 理解回答角度
- "你是一个有10年经验的产品经理"

**2. 任务**（必须）
```
❌ 写代码
✅ Python函数，接收列表，返回最大最小值
```

**3. 要求**（可选但推荐）
- 长度：100字以内、不超过3段
- 格式：表格、bullet points、JSON
- 风格：口语化、正式、轻松
- 受众：小学生、专业人士

**4. 示例**（高级）
```
输入：今天很开心
输出：🎉 今天很开心！

输入：考试没考好
现在请处理：收到了offer
```

## 例子

**模糊**：`帮我写封信`

**清晰**：
```
角色：专业商务秘书
任务：写给客户的道歉信
要求：语气诚恳但不卑微，200字，解释供应商问题，承诺3天内解决
```

## 高级技巧

### 思维链
```
请逐步分析：先理解问题 → 分析关键点 → 给出结论
```

### 结构化输出
| 格式 | 适用场景 |
|------|---------|
| 表格 | 对比分析 |
| JSON | 程序集成 |
| Markdown | 报告文档 |

### 多步骤分解
```
帮我完成大任务，分三步：
第一步... 第二步... 第三步...
```

## 常见错误

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| "帮我看看" | "检查这段代码安全问题" |
| "写长一点但要简短" | "3句话解释" |
| "这个方案怎么样" | "评估营销方案：预算10万，目标..." |
| "帮我看看这个方案" | "分析营销方案：预算5万，人群大学生，产品零食" |
