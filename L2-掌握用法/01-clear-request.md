# 清晰表达需求

## 问题出在哪

同样问题效果不同，原因是 **prompt 太模糊**。

## 公式：角色 + 任务 + 要求

```
角色：（可选）你希望 AI 扮演什么
任务：具体要做什么
要求：限制、格式、风格
```

### 例子

**模糊**：`帮我写封信`

**清晰**：
```
角色：专业商务秘书
任务：写给客户的道歉信
要求：语气诚恳但不卑微，200字，解释供应商问题，承诺3天内解决
```

## 三要素详解

**1. 角色**（可选）
- 老师、翻译、编辑、同学...
- 帮助 AI 理解回答角度

**2. 任务**
```
❌ 写代码
✅ Python函数，接收列表，返回最大最小值
```

**3. 要求**
- 长度：100字以内、不超过3段
- 格式：表格、bullet points
- 风格：口语化、正式、轻松
- 受众：小学生、专业人士

## 常见错误

**假设 AI 知道你的情况**
```
❌ 帮我看看这个方案
✅ 分析营销方案：预算5万，人群大学生，产品零食
```

**要求不明确**
```
❌ 写好一点
✅ 润色：语言生动，逻辑清晰，500字以上
```

**一次要求太多**
```
❌ 写论文：摘要、引言、方法、结论、参考文献、查重
→ 拆成多个问题
```

## 练一练

把模糊 prompt 改清晰：

1. "教我编程"
2. "帮我写总结"
3. "看看这篇文章"
