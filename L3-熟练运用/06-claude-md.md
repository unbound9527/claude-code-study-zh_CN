# Claude.md 的作用

## 什么是 Claude.md

项目根目录的**项目说明书**。Claude Code 打开项目时自动读取。

```
项目/
├── CLAUDE.md  ← 这个！
├── src/
└── README.md
```

## 作用

| 作用 | 说明 |
|------|------|
| 提供背景 | 让 AI 了解项目 |
| 设置规则 | 编码规范、流程 |
| 减少重复 | 不用每次解释 |
| 保持一致 | 每次回答符合项目 |

## Claude.md vs 对话说明

| | Claude.md | 对话中说明 |
|--|-----------|------------|
| 持久性 | 永久 | 每次都要说 |
| 范围 | 当前项目 | 仅当前对话 |

## 基本结构

```markdown
# 项目名称

## 项目背景
[做什么的]

## 技术栈
- 前端：...
- 后端：...

## 代码规范
[规范]

## 工作流程
[流程]
```

## vs .claude 目录

| 文件 | 作用 |
|------|------|
| `CLAUDE.md` | 项目级 |
| `~/.claude/settings.json` | 全局 |
