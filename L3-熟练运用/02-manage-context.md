# 管理对话上下文

## 上下文是什么

AI 能记住的对话内容。有容量限制，对话太长会忘记之前说的。

## 上下文不足的表现

- AI 开始"忘记"
- 回答质量下降
- 建议"压缩"

## 管理技巧

**简洁提问**：
```
❌ "之前A说了，B也提到，然后C..."
✅ "继续说C"
```

**开启新对话**：话题转换时

**定期总结**：
```
"总结之前的要点，然后继续"
```

## /compact 命令

| 命令 | 效果 |
|------|------|
| `/compact` | 轻度压缩：保留关键结论 |
| `/compact full` | 中度压缩：只保留决策点 |
| `/compact summary` | 深度压缩：只保留高层摘要 |

## 什么时候开新对话

- 话题转换
- 任务完成
- AI 开始"失忆"
- 项目太大，按模块分开
