# Claude Code 中文教程

> 从零开始，系统学习 Claude Code，成为 AI 协作高手。

---

## 这是什么？

一套完整的学习路径，帮你从"第一次用 AI"到"熟练使用 Claude Code"。

**特别适合**：对 AI 工具感兴趣的任何人。无论你是刚接触命令行的技术新手，还是想深度定制 Agent 的资深架构师。

---

## 学习路径

```
L0 环境配置 → L1 认识工具 → L2 掌握用法 → L3 熟练运用 → L4 精通技巧 → L5 工程化学习 → L6 实践经验
```

| 阶段 | 简介 | 目标 |
|------|------|------|
| **L0 环境配置** | 安装教程、代理设置、国产模型接入 | 零基础入门 |
| **L1 认识工具** | AI 基础、第一次对话、Git、CLI、插件生态 | 会基本操作 |
| **L2 掌握用法** | 清晰表达、工具使用、语音输入、Prompt 技巧 | 能用得好 |
| **L3 熟练运用** | 任务分解、上下文管理、迭代优化、工作流、Claude.md | 用得巧 |
| **L4 精通技巧** | 记忆、子代理、Agent Teams、MCP、Hooks、安全、权限 | 精通高级功能 |
| **L5 工程化学习** | 源码解读、从零构建 Agent、MCP 协议、Hooks 机制 | 理解原理并实现 |
| **L6 实践经验** | 高效使用模式、常见陷阱与避坑、进阶用户案例 | 最佳实践汇总 |
