# 文档润色 Skill

## 用途

对 Markdown 文档进行语言润色，提升可读性和表达质量：
- 病句修改
- 冗余表达简化
- 标点符号规范化
- 术语一致性
- 格式统一

## 相关开源项目

| 项目 | Stars | 说明 |
|------|-------|------|
| [textlint](https://github.com/textlint/textlint) | ⭐ 3k+ | 可配置的文本检查工具 |
| [write-good](https://github.com/btford/write-good) | ⭐ 4k+ | 英文写作检查工具 |

## 使用方式

```
/polish <文件路径>
```

## 润色项目

### 1. 语言质量
- 病句识别与修改
- 重复表达精简
- 歧义表达优化

### 2. 标点规范
- 中英文标点混用修正
- 多余空格清理
- 行末多余空格删除

### 3. 术语一致性
- 检查文档内同一术语的用法
- 建议统一的术语表达

### 4. 格式统一
- 标题大小写规范
- 列表符号一致性
- 代码块标记统一

## 输出格式

```
## 文档润色报告

### 修改文件：xxx.md

#### 语义优化（N 处）
- [行 N] 原文 → 修改后

#### 冗余精简（N 处）
- [行 N] 原文 → 修改后

#### 标点修正（N 处）
- [行 N] 原文 → 修改后

---
总计：M 处修改
```
