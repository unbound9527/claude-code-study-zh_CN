# README 更新 Prompt

## 任务

扫描 `docs/` 目录并更新 `README.md` 的目录导航。

## 工作流程

1. **扫描目录结构**
   ```
   docs/
   ├── L1-认识工具/
   │   ├── 01-what-is-ai.md
   │   └── ...
   ├── L2-掌握用法/
   │   └── ...
   ```

2. **提取文档信息**
   - 从每个 `.md` 文件读取第一行作为标题
   - 按目录组织结构

3. **生成目录格式**
   ```markdown
   ### L1 认识工具

   - [标题](docs/L1-认识工具/文件名.md)
   ```

4. **更新 README.md**
   - 定位目录区域（以 "## 快速导航" 开始）
   - 替换现有目录内容
   - 保持其他内容不变

## 目录结构规范

```markdown
## 快速导航

### L1 认识工具
- [标题](docs/L1-认识工具/文件名.md)
- ...

### L2 掌握用法
- [标题](docs/L2-掌握用法/文件名.md)
- ...

（依此类推 L3-L6 和附录）
```

## 注意事项

- 保持原有格式和顺序
- 不要修改其他内容
- 只更新"快速导航"部分
- 附录部分使用不同的文件名模式
