# 链接检查 Prompt

## 任务

检查指定目录下所有 Markdown 文件的链接完整性。

## 检查规则

1. **相对 .md 链接**：验证链接目标文件存在
2. **相对路径解析**：相对于源文件位置解析路径
3. **外部 URL**：检查格式有效性（记录但不验证可访问性）

## 实现步骤

1. 使用 glob 扫描目录下所有 `*.md` 文件
2. 对每个文件，使用 grep 提取所有 Markdown 链接 `[text](url)` 格式
3. 对每个链接：
   - 如果是 `.md` 文件，检查目标是否存在
   - 如果是相对路径，按源文件位置解析
   - 如果是外部 URL，记录格式
4. 生成问题报告

## 输出格式

```markdown
### 链接检查结果

**扫描文件数**: N
**发现问题数**: M

#### 断链列表

| 源文件 | 行号 | 链接 | 问题 |
|--------|------|------|------|
| xxx.md | 15 | [link](target.md) | 目标文件不存在 |
```

#### 外部链接（格式正确）

| 源文件 | 链接 |
|--------|------|
| xxx.md | https://example.com |
```
