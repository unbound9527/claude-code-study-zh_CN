# 标题层级分析 Prompt

## 任务

检查 Markdown 文件的标题层级是否符合规范。

## 规范定义

1. 每个文件只能有一个 H1（# 标题）
2. 标题层级必须连续，不能跳级
3. 允许的层级顺序：H1 → H2 → H3 → H4（最多到 H4）

## 检查规则

对于每个文件：
1. 读取文件内容
2. 记录所有标题及其层级（行号）
3. 检测以下问题：
   - 是否存在多个 H1
   - 是否有层级跳跃（如 H1 后直接 H3，中间缺少 H2）
   - 是否使用了 H5 或更深的标题
   - H1 位置是否在文件开头（前 10 行内）

## 输出格式

```markdown
### 标题层级检查结果

**扫描文件数**: N
**发现问题数**: M

#### 问题列表

| 文件 | 位置 | 当前层级 | 问题描述 |
|------|------|----------|----------|
| xxx.md | 行 15 | # | 多个 H1 标题（首个在行 1） |
| xxx.md | 行 30 | ### | H1 后跳过 H2 |
| xxx.md | 行 45 | ##### | 不允许使用 H5 及更深层级 |
```
